# opar-server
server app for opar

npm list
```sh
	async
	body-parser
	express
	jwt
	mongoose
```

required software
+ [mongodb](http://www.mongodb.com) v3.0.x
+ [nodejs](httts://nodejs.org) v4.2.2 LTS

clone : 
```sh
	$ git clone https://github.com/froztic/opar-server.git
```

setup : 
```sh
	$ cd opar-*
	$ sudo npm install
```

run : 
```sh
	$ node app.js
```

default port :
```sh
	app 	: 4200
```
